# flexdashboard

-  Utiliza R Markdown para publicar un grupo de visualizaciones de datos relacionadas como panel de control.   
-  Soporte para una amplia variedad de componentes, incluyendo htmlwidgets; gráficos base, en lattice y en cuadrícula (grid graphics); datos tabulares; calibres (gauges) y cajas de valores (value boxes); y anotaciones de texto.   
-  Flexible y fácil de especificar en layouts basados en filas y columnas. Los componentes se redimensionan inteligentemente para llenar el navegador y se adaptan para su visualización en dispositivos móviles.   
-  Layouts de storyboard para presentar secuencias de visualizaciones y comentarios relacionados.   
-  Opcionalmente, usa Shiny para realiar visualizaciones dinámicamente.   
- Opcionalmente, usa bslib para personalizar fácilmente los colores principales, las fuentes y más.
